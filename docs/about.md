## 关于网站

[VitePress](https://vitepress.dev/zh/) 使用 Markdown 编写的源内容，生成可以轻松部署在任何地方的静态 HTML 页面。

本站就是使用 VitePress 搭建，网站托管在由 [Cloudflare](https://www.cloudflare-cn.com/) 免费提供的静态空间内，如遇网站打开缓慢，可刷新网页或稍后再试！如遇图片加载失败，大概率是因为图片服务器关机所致，请与我联系！
